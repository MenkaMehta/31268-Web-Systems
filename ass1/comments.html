Technicalities

All the web pages in this website contain four main div classes:

  1. class "wrapper" - This is the class that contain all the other div classes.
  2. class "header" - This is the class that contains the main heading of the page and the links that navigate to the other web pages
  3. class "content" - This is the class contains all the information required for that particular page.
  4. class "footer" - This is the class that is located at the bottom of the page representing the "copyright © 2017 Menka Mehta"
